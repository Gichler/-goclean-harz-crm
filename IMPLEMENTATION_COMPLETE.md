### 🚀 So starten Sie das System:

#### 1. Backend starten:
```bash
python simple_backend.py
```
Das Backend läuft dann auf: http://localhost:5000

#### 2. Frontend starten:
```bash
npm run dev
```
Das Frontend läuft dann auf: http://localhost:5173
